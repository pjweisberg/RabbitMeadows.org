Web Colors

This page was generated by the FrontPage Corporate Presence Web Wizard.

All of the other pages in the web refer to this page for their color settings. The shared attributes (and their HTML tags) are:

  • the background image (BACKGROUND)
  • the background color (BGCOLOR)
  • the color of the text (TEXT)
  • the color of a link before it has been visited (LINK)
  • the color of a link after it has been visited (VLINK)
  • the color of a link when the user clicks on it (ALINK)

Use the Page Properties dialog to change the color settings on this page. When you save the page, all of the other pages referring to it will automatically be updated to use the same color settings.
